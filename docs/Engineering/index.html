工程工具与排障

这一栏聚合命令行工具、Xcode 工具链、Git 排障和日常工程效率内容。目标不是讲原理，而是解决“遇到问题时怎么快速处理”。

适合谁看

  • 经常需要排查本地环境和线上问题的开发者
  • 想补齐 Git、命令行和 Xcode 工具链常识的人
  • 更关注效率和落地方法，而不是纯理论的人

推荐阅读顺序

  1. 先看命令与环境，掌握排障的基础工具。
  2. 再看开发工具，补齐日常工作流。
  3. 最后看 Git 与其他补充专题，提升整体效率。

# 行业研究

这一栏用于把公司研究放回行业背景中看。重点是理解行业商业模式、监管约束、盈利驱动和风险暴露，而不是只盯单家公司。

## 适合谁看

- 想建立行业视角的投资者
- 需要快速回顾地产、银行、保险逻辑的人
- 想把宏观、财报和公司研究串起来的人

## 推荐阅读顺序

1. 先看行业概览，理解商业模式和监管环境。
2. 再看银行等高权重行业，建立财报阅读重点。
3. 最后把行业特性和估值方法结合起来使用。

计算机基础

这部分收录网络协议、数据结构、算法和数据库基础内容。适合从零梳理知识框架，也适合在写代码时快速回查概念。

适合谁看

  • 想补齐计算机基础的开发者
  • 需要快速回顾协议与数据库原理的工程师
  • 想按主题系统复习而不是零散搜索的人

推荐阅读顺序

  1. 先看网络协议，建立请求、连接和传输层的整体认知。
  2. 再看数据结构与算法，补齐编码时最常用的抽象能力。
  3. 最后看 MySQL，理解数据存储与查询优化。

投资方法

这一栏聚焦投资人物、方法论和宏观框架，适合先建立判断体系，再进入财报与行业分析。重点不是背观点，而是吸收可复用的分析框架。

适合谁看

  • 想系统学习价值投资方法的人
  • 需要从人物案例中提炼投资框架的读者
  • 关注宏观流动性与资产配置关系的人

推荐阅读顺序

  1. 先看投资框架类文章，建立基本判断方法。
  2. 再看经典投资人物，总结他们的共同原则与差异。
  3. 最后看货币和信用相关内容，补齐宏观视角。
